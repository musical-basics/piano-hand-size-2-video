# Pass 3 Paper Edit

Status: rough ordered edit. This turns the Pass 2 moment pool into a concrete story shape, but it is not yet frame-level assembly notes.

Target length: about 11:30 to 12:30.

Core promise:

> I drove overnight to pick up rare DS 6.0 and DS 5.5 piano keyboards because hand size changes the way you experience the piano.

## Editorial Shape

The best version is a road quest that gradually becomes an argument.

The first half should make the trip feel real: the late hour, snacks, fatigue, Pennsylvania roads, David's factory, and the physical keyboards.

The second half should answer why any of this matters: smaller keys are not just about reaching bigger intervals; they can change technique, tension, practice efficiency, and whether the instrument fits the body.
