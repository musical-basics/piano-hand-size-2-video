# Music Bed Notes

`pass8_travel_bed.wav` is an original procedural ambient bed generated locally with FFmpeg for this project.

It does not sample or quote an external track. It is intended as royalty-free placeholder music for the travel/narration sections of the rough cut.

Current usage:

- `make_rough_review_cut_v5.sh`
- low under title cards
- lower under narration voiceovers
- not placed under clips where Lionel is speaking on camera

The WAV is ignored by git because it is generated media. If it is missing,
`make_rough_review_cut_v5.sh` regenerates it before rendering.
